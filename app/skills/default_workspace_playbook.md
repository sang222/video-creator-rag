# Default Workspace Playbook

Principles:
- Validate buyer intent before scaling production cost.
- Reuse proven assets and formats before generating new media.
- Preserve channel voice and avoid spam-like repetition.
- Favor concise scripts with a clear hook, useful body, and soft CTA.

Cost policy:
- New or low-confidence channels run in monetization validation mode.
- Premium production requires explicit human pre-approval.
